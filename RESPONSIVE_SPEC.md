# 响应式设计规范

## 断点
- 手机: `<768px`（max-width: 48em）
- 平板: `768px - 1024px`（48em - 64em）
- 桌面: `>1024px`（min-width: 64em）

## 导航
- 桌面: 左侧品牌与链接横向展示；搜索居中；右侧用户与主题开关
- 平板: 搜索区扩展至中间区域；链接保持横向但间距缩小
- 手机: 隐藏横向链接，显示汉堡菜单；移动菜单容器在导航下方展开

## 搜索与分类
- 桌面: 主类横向排列；子类下拉定位于主类下方、双列胶囊
- 平板: 双列胶囊，间距缩小
- 手机: 主类可横向滚动；子类变为单列，定位改为静态堆叠

## 排版
- 根字体: `html { font-size: clamp(14px, 1.2vw, 16px) }`
- 标题: 使用 `clamp` 按需定义（示例：`clamp(1rem, 2.5vw, 1.5rem)`）
- 行高: 1.4 - 1.6
- 颜色分类：显示颜色圆点图标，其它胶囊不加图标

## 间距
- 使用 `rem` 定义：`0.5rem / 0.75rem / 1rem / 1.5rem / 2rem`
- 手机：整体间距缩小 20%-30%

## 媒体元素
- 图片/视频：宽度 100%，高度自适应；容器圆角在移动端略小
- 懒加载：保持

## 动画
- 过渡：`transition: all 0.2s - 0.5s`，使用 `cubic-bezier(0.25, 1, 0.5, 1)`
- 移动端减少复杂阴影以提升性能

## 暗色模式
- 子类胶囊背景：`#111827`，边框透明白 0.15，文字 `#e2e8f0`
- 悬停背景：`#1e293b`

## 清晰度（高分屏）
- 文本抗锯齿：`-webkit-font-smoothing: antialiased; -moz-osx-font-smoothing: grayscale;`
- 图片使用高分辨率来源；缩放使用 `height: auto`

## 测试清单
- iPhone 12/14 (Safari/Chrome)、Android 主流设备 (Chrome)
- iPad (Safari)、Windows 桌面 (Chrome/Edge)、macOS (Safari/Chrome)
- 断点切换时导航、搜索、分类胶囊、壁纸网格均无跳闪
